# Structure

Directory layout and organization for `gomoku-ai`.

## Key Locations

| Path | Description |
|------|-------------|
| `index.html` | Application entry point and UI structure. |
| `style.css` | Global styling and dark mode definitions. |
| `src/` | **Source Code** |
| `src/main.js` | Main application logic, UI, and rendering. |
| `src/gomoku.js` | Game engine and rule definitions. |
| `src/ai.js` | AI bot implementation. |
| `src/i18n.js` | Localization dictionary and helper functions. |
| `tests/` | **Testing Suite** |
| `tests/*.test.js` | Unit tests for engine and AI logic. |
| `package.json` | Project metadata and dependency configuration. |

## Naming Conventions

- **Files**: Lowercase with hyphens or underscores (e.g., `gomoku.js`).
- **Directories**: Lowercase (e.g., `src`, `tests`).
- **JavaScript**: CamelCase for variables/functions, UPPER_SNAKE_CASE for constants.
